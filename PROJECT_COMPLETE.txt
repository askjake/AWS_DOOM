╔══════════════════════════════════════════════════════════════════════════════╗
║                              WHAT THE GAME DOES                              ║
╚══════════════════════════════════════════════════════════════════════════════╝

AWS DOOM transforms your AWS infrastructure into a playable 3D dungeon:

  🏢 VPCs         → Large blue container rooms
  📦 Subnets      → Cyan rooms within VPCs
  🔒 Security Gps → Orange locked areas (need keys)
  ⚙️  EKS Clusters → Purple major zones
  🔀 Load Balancers → Green entry points
  💾 RDS Databases → Red data vaults
  📁 S3 Buckets   → Yellow storage areas
  🚪 Hallways     → Gray network connections

You navigate this 3D world in first-person (like classic DOOM) and collect
"keys" to unlock secured resources. Real AWS metadata displays in the HUD.

╔══════════════════════════════════════════════════════════════════════════════╗
║                           HOW TO PLAY (3 STEPS)                              ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. INSTALL PYGAME:
   $ pip install pygame>=2.5.0

2. VERIFY SNAPSHOTS EXIST:
   $ ls ~/AWS_Architecture_explorer/snapshots/*.json

3. LAUNCH GAME:
   $ cd ~/AWS_Architecture_explorer/AWS_DOOM/game
   $ ./run_aws_doom.sh

That's it! The game will open in a 1280x720 window.
