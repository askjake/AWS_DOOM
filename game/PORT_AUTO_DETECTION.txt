║                          CODE CHANGES                                        ║
╚══════════════════════════════════════════════════════════════════════════════╝

NEW FUNCTION: find_available_port()
  - Tries ports in range (default: 5000-5019)
  - Tests each port by attempting to bind
  - Returns first available port
  - Returns None if all ports busy

NEW FUNCTION: get_local_ip()
  - Determines local network IP address
  - Used to display network access URL
  - Fallback to 'localhost' if detection fails

UPDATED: main()
  - Calls find_available_port() before starting server
  - Displays both local and network URLs with actual port
  - Handles case where no ports available (error message)
  - Server binds to dynamic port instead of hardcoded 5000

UPDATED: JavaScript Client
  - Automatically connects to server's port
  - Displays current port in connection status
  - No hardcoded port references

To change port range, edit aws_doom_web.py:

  # Line ~360
  server_port = find_available_port(
      start_port=5000,      # Start at port 5000
      max_attempts=20       # Try 20 ports (5000-5019)
  )

Custom port range examples:
  find_available_port(start_port=8000, max_attempts=10)  # 8000-8009
  find_available_port(start_port=3000, max_attempts=50)  # 3000-3049
